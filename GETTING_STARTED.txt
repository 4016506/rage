==================================================================
                    RAGE METER - GETTING STARTED
==================================================================

Welcome to your Rage Meter! Follow these simple steps to get raging.

==================================================================
STEP 1: INSTALL DEPENDENCIES
==================================================================

Open terminal in this directory and run:
  npm install

==================================================================
STEP 2: SET UP FIREBASE (IMPORTANT!)
==================================================================

You MUST configure Firebase before the app works!

1. Go to: https://console.firebase.google.com/
2. Create a new project
3. Enable Firestore Database
4. Enable Authentication (Email/Password)
5. Create a user account
6. Copy Firebase configuration

See FIREBASE_SETUP.md for detailed step-by-step instructions.

==================================================================
STEP 3: ADD FIREBASE CONFIG
==================================================================

Open: src/config/firebase.ts
Replace the placeholder values with your Firebase config

==================================================================
STEP 4: START THE APP
==================================================================

Run: npm run dev

Open browser to: http://localhost:5173

==================================================================

==================================================================
